# Deep RL experiments

A home for my deep reinforcement learning experiments. The aim of this repository is to
gradually work through existing reinforcement learning methods and then eventually
recreate state-of-the-art papers and my own ideas.
I'm going to hold myself to close-to-professional level research standards in this
repository: emphasis here lies in producing full experimental analysis. 

That means that 
1. all algorithms used will provide full training curves and demonstrate reproducibility,
2. logging will always be integrated and
4. all outputs will be fully tested before being commited.
